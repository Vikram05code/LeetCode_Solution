# HashMap Pattern

##  When to Use
Use a HashMap when you need quick lookups, especially in problems where you're trying to find pairs or check if a value already exists.

##  Key Idea
While iterating through the array, store elements in a HashMap and keep checking if the required value (like a complement) is already present.

##  Example
- 1 - Two Sum

##  Complexity
- Time: O(n) — we go through the array once
- Space: O(n) — we store elements in the map
